# SQLSaturday #900 - Vitória 2019
Event Date: 24-08-2019 - Session time: 09:00:00 - Track: Enterprise Database Administration  Deployment
## Speaker: Igor Santos de Paula
## Title: Relatos das trincheiras: O dia a dia de uma consultoria de banco de dados
## Abstract:
### Nessa sessão iremos abordar os principais problemas arquiteturais, e suas soluções, que encontro nas mais diversas corporações brasileiras. Desde bancos de dados recebendo 100% de querys Adhoc, CPDs inundados, até servidores que foram metralhados e não possuiam backup.
Falaremos sobre arquitetura de dados, boas práticas de backup, alta disponibilidade, disaster recovery, performance, boas práticas de configuração e etc.
