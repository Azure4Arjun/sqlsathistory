# SQLSaturday #570 - São Paulo 2016
Event Date: 08-10-2016 - Session time: 15:00:00 - Track: Big Data e Data Science
## Speaker: Rodrigo Dornel
## Title: SQL Server Heterogêneo - Como integrar meu banco de dados relacional com NoSQL e BigData
## Abstract:
### Desde que o SQL Server começou a suportar XML já sabíamos que estávamos entrando em uma nova era dos sistemas de banco de dados. Com a consolidação das arquiteturas em Nuvem e Big Data os sistemas tradicionais estão cada vez mais integrados e interagindo com sistemas não estruturados. 
Nessa sessão o MVP Rodrigo Dornel irá mostrar como integrar o SQL Server com novas tecnologias, através de PolyBase, External Tables, JSON, SSIS e Stretch Database.
