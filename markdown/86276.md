# SQLSaturday #811 - Rio de Janeiro 2018
Event Date: 24-11-2018 - Session time: 16:00:00 - Track: Strategy and Architecture
## Speaker: Danielle Monteiro
## Title: 10 motivos para se preocupar com a Lei Geral de Proteção de Dados
## Abstract:
### A Lei Geral de Proteção de Dados entrará em vigor em fevereiro de 2020, e vai mudar o modo como as organizações "cuidam" dos dados. 
Você conhece esta lei? Sabe quais os impactos que ela trás para o seus projetos? Como o SQL Server pode ser usado para garantir que seus dados estejam tão seguros quanto a lei exige?
