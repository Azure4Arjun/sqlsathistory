# SQLSaturday #900 - Vitória 2019
Event Date: 24-08-2019 - Session time: 11:00:00 - Track: Application  Database Development
## Speaker: Rafael dos Santos
## Title: Aliviando o Stress do DBA (e do seu banco de dados!)
## Abstract:
### Através de padrões de desenvolvimento, Rafael dos Santos, vai ensinar como evitar os principais gargalos em aplicações que atendem milhares de usuários e como deixar seu banco de dados mais estável.
