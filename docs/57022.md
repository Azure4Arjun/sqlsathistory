# SQLSaturday #598 - Costa Rica 2017
Event Date: 29-04-2017 - Session time: 10:15:00 - Track: Strategy and Architecture
## Speaker: Stephan Torres
## Title: Como ser victima de un cyberataque y poder contar la historia
## Abstract:
### Todos estamos expuestos a distintos tipos de riesgos informáticos, cada vez más servicios son críticos y mantener nuestra operación "en línea" es crucial.
En esta charla abordaremos los aspectos más relevantes de cómo asegurar nuestros servidores de base de datos en Windows y Linux, además de cómo establecer estrategias de mitigación y recuperación en caso de ser víctima de un ciberataque.
Además, hablaremos un poco de estructuras de datos inmutables o persistentes, estilo blockchain.
