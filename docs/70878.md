# SQLSaturday #703 - Costa Rica 2018
Event Date: 21-04-2018 - Session time: 11:30:00 - Track: Application  Database Development
## Speaker: Adrian Miranda
## Title: Cosas que tienes que hacer antes de llamar a Microsoft Support
## Abstract:
### Como hacer troubleshooting de nuestro SQL Server sin tener que llamar a Microsoft Support? Mostraremos técnicas, puntos escenciales que debes de chequear antes de crear un caso de soporte. Muchos de los casos los podemos resolver nosotros mismos, solo debemos de tener conciencia de donde buscar y como buscar. Todas estas técnicas las abarcaremos en esta sesión.
