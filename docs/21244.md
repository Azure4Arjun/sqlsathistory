# SQLSaturday #325 - São Paulo 2014
Event Date: 27-09-2014 - Session time: 00:00:00 - Track: Administração
## Speaker: Marcus Vinícius Bittencourt
## Title: MythBusters - Caçadores de Mitos
## Abstract:
### Baseado na série DBA Myth A Day de Paul Randal e também na proposta e descontração de MythBusters.Esta sessão pretende revelar o que há por trás dos mitos do SQL Server.Colocá-los a provas de fogo, testes, reproduções, validações até obtermos as explicações para cada um deles e revelar se o mito é verdadeiro ou pode ser quebrado.
