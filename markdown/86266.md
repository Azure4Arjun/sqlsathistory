# SQLSaturday #811 - Rio de Janeiro 2018
Event Date: 24-11-2018 - Session time: 15:00:00 - Track: Strategy and Architecture
## Speaker: Wiluey Sousa
## Title: Migrando seus pacotes SSIS para a nuvem com o Azure Data Factory
## Abstract:
### Você migrou seus bancos de dados SQL Server on-premises para o Azure, porém executava em seu ambiente alguns processos do SSIS e agora?
Nesta palestra entenderemos como configurar seus pacotes SSIS com o Data Factory, utilizando de maneira efetiva os recursos da ferramenta.
Também conheceremos o uso de Triggers, Pipelines e Linked Services na criação do processo.
