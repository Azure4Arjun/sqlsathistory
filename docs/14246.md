# SQLSaturday #325 - São Paulo 2014
Event Date: 27-09-2014 - Session time: 00:00:00 - Track: Administração
## Speaker: Marcelo Fernandes
## Title: Utilizando e monitorando o CLR e Broker no SQL Server
## Abstract:
### O SQL Serve é um produto fantástico com diversas features e algumas não tiramos proveito de algumas features por não conhecê-las.
Nesta sessão você conhecerá o CLR e o Broker e como construir procedures com código .NET com segurança e performance e como monitorar estas fatures no SQL Server.
