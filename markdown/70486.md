# SQLSaturday #690 - Lima 2017
Event Date: 02-12-2017 - Session time: 12:50:00 - Track: Big Data  Analytics
## Speaker: Alejandro Humberto Sánchez Núñez
## Title: Hybrid BI con Power BI Report Server y PowerBI.com
## Abstract:
### El concepto de la charla es guiar de porqué existe PBI RS y mostrar la diferencia con powerbi.com (servicio nube); ademas de por que usar PBI RS, cuando es recomendable ir a por esta solución y como sacarle a las caracteristicas que van siendo habilitadas para esta versión. 

Punto aparte no olvidar que si se usa PBI RS y quieres ir al servicio, puedes hacer cuando se necesite o la organización lo requiera.
