# SQLSaturday #676 - São Paulo 2017
Event Date: 30-09-2017 - Session time: 16:30:00 - Track: Database Administration  Deployment
## Speaker: Fabio Cotrim
## Title: Monitorando o seu ambiente SQL Server com o Zabbix
## Abstract:
### Com a utilização da ferramenta open source ZABBIX, seus templates e seus plugins, é possível monitorar vários fatores do seu ambiente de banco de dados SQL Server, identificando os ofensores de performance, prováveis falhas, etc.

Pretendo mostrar quais as vantagens de utilizar-se um produto de monitoração externo ao seu ambiente, ao invés de utilizar recursos internos do Windows e do SQL Server.
