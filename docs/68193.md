# SQLSaturday #676 - São Paulo 2017
Event Date: 30-09-2017 - Session time: 12:00:00 - Track: Cloud Computing
## Speaker: Renato Groffe
## Title: Tecnologias de bancos de dados no Azure: conhecendo as opções relacionais e NoSQL disponíveis
## Abstract:
### Utiliza bancos de dados relacionais ou NoSQL nas suas aplicações? Gostaria de migrar ou mesmo desenvolver algo do zero e sem grandes complicações na nuvem? Acompanhe então esta apresentação para conhecer algumas das opções disponibilizadas como serviço pelo Microsoft Azure.
