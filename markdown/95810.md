# SQLSaturday #906 - São Paulo 2019
Event Date: 28-09-2019 - Session time: 10:00:00 - Track: Application  Database Development
## Speaker: Marcel Inowe
## Title: Query Performance vs ORM
## Abstract:
### Sabemos o quanto os ORM(Object Relational Mapper) como NHynernate e Entity Framewor ajudam no desenvolvimento das aplicações, mas como fica a performance das queries no banco de dados? 
O que acontece quando aquela aplicação que era pequena começa a crescer, crescer e ficar lenta?
Nessa sessão abordaremos esse assunto tão polêmico, quanto interessante e mostraremos algumas técnicas de como podemos ajudar os desenvolvedores a alcançarem a performance perfeita em seu banco de dados.
