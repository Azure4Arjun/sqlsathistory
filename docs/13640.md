# SQLSaturday #322 - Guatemala 2014
Event Date: 26-07-2014 - Session time: 00:00:00 - Track: Track 2
## Speaker: Eduardo Castro
## Title: Introducción al análisis predictivo con SQL Server
## Abstract:
### En esta sesión se presentan los conceptos fundamentales de análisis predictivo y cómo se implementan con SQL Server y las herramientas de análisis de datos tales como Excel. 

Adicionalmente, se muestra cómo se puede publicar dichos análisis por medio de SharePoint.
