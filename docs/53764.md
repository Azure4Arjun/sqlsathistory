# SQLSaturday #570 - São Paulo 2016
Event Date: 08-10-2016 - Session time: 11:30:00 - Track: Big Data e Data Science
## Speaker: Thiago Zavaschi
## Title: Advanced Analytics using SQL Server R Services
## Abstract:
### Nesta sesão vamos ver uma introdução da linguagem mais popular no meio dos cientistas de dados e também no Microsoft R Server, integrado ao SQL Server sob o nome de R Services.
R é uma linguagem aberta com foco em análise estatística e está integrada nativamente ao SQL Server 2016, sob o nome de SQL Server R Services.
