# SQLSaturday #608 - Bogota 2017
Event Date: 13-05-2017 - Session time: 11:30:00 - Track: BI/BA
## Speaker: Raul Martin Sarachaga Diaz
## Title: Explorando los sabores de SQL Server Integration Services 2016
## Abstract:
### En este Sesión se hablará sobre como se desarrolla un Proyecto de Business Intelligence, para luego presentar todas las nuevas caracteristicas que tiene SSIS 2016, tocando  algunas cosas interesantes de versiones anteriores y mejores prácticas en el desarrollo de proyectos ETL.
