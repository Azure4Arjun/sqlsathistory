# SQLSaturday #703 - Costa Rica 2018
Event Date: 21-04-2018 - Session time: 14:45:00 - Track: Application  Database Development
## Speaker: Carlos Rojas Vargas
## Title: Aprovechando los AlwaysOn Basic Availability Groups en SQL-Server Standard Edition
## Abstract:
### En esta session aprenderemos sobre la Implementación y Configuración de los AlwaysOn Basic Availability Groups en las versiones Standard de SQL-Server, veremos cómo aprovechar sus funcionalidades para Alta Disponibilidad y también cómo combinar características de SQL-Server como el Snapshot con los Basic Availability Groups para tener un nivel básico de balanceo de cargas. Adicionalmente veremos cómo las aplicaciones pueden aprovechar estas características.
