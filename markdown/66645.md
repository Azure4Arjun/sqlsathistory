# SQLSaturday #676 - São Paulo 2017
Event Date: 30-09-2017 - Session time: 14:00:00 - Track: Database Administration  Deployment
## Speaker: Marcel Inowe
## Title: Analisando query performance com e sem o Query Store.
## Abstract:
### Query Store é um ótimo recurso para análise de problemas de performance de query que foi lançado no SQL Server 2016. 
Nessa sessão mostraremos o que é o Query Store, como analisar e resolver alguns problemas de performance de query com e sem o Query Store, afinal nem todos os ambientes SQL Server que administramos estão no SQL Server 2016.
