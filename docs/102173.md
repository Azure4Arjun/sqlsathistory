# SQLSaturday #941 - Joinville - Virtual 2020
Event Date: 03-10-2020 - Session time: 09:30:00 - Track: Application  Database Development
## Speaker: Marcel Inowe
## Title: Porque minha aplicação com Entity Framework está lenta?
## Abstract:
### Sabemos o quanto os ORM(Object Relational Mapper) como o Entity Framework ajudam no desenvolvimento das aplicações, mas como fica a performance das queries no banco de dados? 
O que acontece quando aquela aplicação que era pequena começa a crescer, crescer e ficar lenta?
Nessa sessão abordaremos esse assunto tão polêmico, quanto interessante e mostraremos algumas técnicas de como podemos ajudar os desenvolvedores a alcançarem a performance perfeita em seu banco de dados.
