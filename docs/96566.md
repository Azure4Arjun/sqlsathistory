# SQLSaturday #906 - São Paulo 2019
Event Date: 28-09-2019 - Session time: 17:00:00 - Track: Analytics and Visualization
## Speaker: Rocio Cairo Presepi
## Title: Como responder mais fácil a suas questões de negócio?
## Abstract:
### Você costuma receber perguntas que não consegue responder? Você sente que possui as informações em sua mão, mas o volume é tão grande que você não sabe por onde começar? O Power BI é sua solução para essas perguntas. O software é usado pelos usuários para responder a perguntas comerciais que anteriormente não eram possíveis perguntas. Nesta sessão, examinaremos as diferentes visualizações e alguns bons exemplos que podem oferecer algumas ideias sobre o potencial real da ferramenta.
