# SQLSaturday #325 - São Paulo 2014
Event Date: 27-09-2014 - Session time: 00:00:00 - Track: Miscellaneous 1
## Speaker: Leandro Ribeiro
## Title: Usando o Hyper-v 2012 para virtualização do SQL Server
## Abstract:
### Virtualizar o SQL Server de alto desempenho é possível?

Nesta sessão será apresentado a arquitetura e as novidades do Hyper-v 2012, os cuidados na configuração para obter o melhor desempenho de um servidor SQL Server e também como escolher e dimensionar o seu hardware para o seu host.
