# SQLSaturday #343 - Medellin 2014
Event Date: 27-09-2014 - Session time: 00:00:00 - Track: Track 3
## Speaker: Andres Felipe Rojas Parra
## Title: Creando soluciones de inteligencia de Negocios con SharePoint 2013
## Abstract:
### La inteligencia de negocios sobre SharePoint 2013 provee un set de herramientas que pueden ser integradas a traves de aplicaciones de Office y otras tecnologías. Con esta charla se pretende mostrar una viariedad de estas herramientas que pueden ayudar a los ejecutivos de las empresas a tomar desiciones mas ajustadas a la realidad y proyectar la hoja de ruta de la compañia.
