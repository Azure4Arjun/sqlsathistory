# SQLSaturday #727 - Mexico 2018
Event Date: 02-06-2018 - Session time: 11:00:00 - Track: Advanced Analysis Techniques
## Speaker: Ismael Medina Muñoz
## Title: Yo quería ser Data Scientist... pero me lastimé la rodilla. Ciencia de datos para mi empresa.
## Abstract:
### En esta sesión se abordará este término de industria del que todos hablan pero que pocos entienden y cuyo roadmap se ve imposible de seguir. Identificaremos algunos de los problemas de industria resueltos con Data Science y serán llevados dentro de este mundo a través de una demo. Identificaremos como usar diferentes herramientas de Microsoft para Data Scientists y entusiastas del Advanced Analytics.
Si, habrá matemáticas... habrá caras de confusión... habrá memes... habrá malos chistes... Pero no van a querer perder ni un minuto de esta sesión!
