# SQLSaturday #672 - Madrid 2017
Event Date: 23-09-2017 - Session time: 13:00:00 - Track: Business Intelligence
## Speaker: Ana Maria Bisbé York
## Title: Escenarios de modelado en DAX, el súper lenguaje de Power BI.
## Abstract:
### En el camino a la creación de buenos cuadros de mandos y modelos analíticos, se impone la necesidad de contar con un buen modelado tabular. En la sesión se analizan y solucionan un grupo de escenarios de modelado con DAX y Power BI, aunque se adaptan a modelos tabulares con SQL Server Analysis Services en SQL Server Data Tools y Power Pivot en Excel. Estos escenarios son los que nos encontramos con frecuencia en proyectos.
