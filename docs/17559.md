# SQLSaturday #325 - São Paulo 2014
Event Date: 27-09-2014 - Session time: 00:00:00 - Track: Miscellaneous 2
## Speaker: João Polisel
## Title: MS DTC de A a Z
## Abstract:
### O que é Microsoft Distributed Transaction Coordinator? Para que ele serve? Como funciona? Como configuro, diagnostico e resolvo problemas?
Se você busca respostas para estas dúvidas, veio ao lugar certo!
Nesta sessão abordarei os princípios de funcionamento do MS DTC, como ele se integra com o SQL Server, como ele se comunica com outras instâncias MS DTC, problemas que podem ocorrer e muito mais!
