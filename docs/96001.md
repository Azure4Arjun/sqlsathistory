# SQLSaturday #906 - São Paulo 2019
Event Date: 28-09-2019 - Session time: 16:00:00 - Track: Application  Database Development
## Speaker: Sergio Passos
## Title: Gerenciamento Mudanças x Versionamento - O que muda no Banco de Dados?
## Abstract:
### Nesta palestra aprenderemos um pouco da importância da gestão de mudanças e por que é preciso versionar o banco de dados, pois durante o ciclo de vida de um software, o banco de dados ou os bancos de dados sofrem mudanças constantes, assim como o seu código-fonte. 
Será feito um comparativo com o objetivo de mostrar as características e diferenças das ferramentas com intuito a servir de base para um decisão de uso entre elas.
