# SQLSaturday #568 - Madrid 2016
Event Date: 24-09-2016 - Session time: 11:35:00 - Track: Sponsor
## Speaker: SQLSaturday Madrid
## Title: Libera el poder de tu aplicativo con Microsoft SQL Server y HPE
## Abstract:
### Data-Driven Enterprise se ha consolidado como una de las estrategias más importantes a seguir por cualquier empresa u organismo público para su futuro inminente. Sin embargo, existen todavía multitud de retos que hacen difícil su implantación en la cadena de toma de decisiones de negocio. En esta presentación descubriremos algunas de las tecnologías que en Hewlett Packard Enterprise creemos pueden, a través de soluciones como Microsoft SQL Server 2016 y Microsoft Windows Server 2016, ayudar a liberar todo el potencial de tus aplicativos para superar dichas barreras.
 
·         Introducción – la empresa ‘dirigida por datos’
·         HPE Superdome X, tecnología para Windows Server 2016 y SQL Server 2016
·         Computación en memoria
·         La memoria como el centro de la arquitectura – Persistent Memory

Juan A. Zapatero (Enterprise Solutions Architect, HPE)
